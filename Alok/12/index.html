Event Properties

type: A string indicating the type of event.
target: A reference to the object that dispatched the event.
currentTarget: A reference to the object that has the event handler attached.
timeStamp: The time when the event was created.
bubbles: Indicates whether the event bubbles up through the DOM tree.
cancelable: Indicates whether the event can be canceled.
eventPhase: An integer representing the current phase of the event flow.
defaultPrevented: Indicates whether the preventDefault() method was called.
isTrusted: Indicates whether the event is trusted.
composed: Indicates whether the event may cross the shadow DOM boundary.
